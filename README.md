# A/B Testing and Statistical Hypothesis Testing in R

## Project Overview
This project demonstrates how statistical hypothesis testing can be used to evaluate the effect of changes between two groups in an A/B test. A/B testing is widely used in business, marketing, and product development to determine whether a change leads to a meaningful improvement.

The goal of this project is to simulate an experiment with two groups and determine whether the observed difference between them is statistically significant.

## Methods Used
- Data simulation for two experimental groups
- Exploratory data analysis
- Two-sample t-test
- Confidence interval estimation
- Visualization of group differences

## Tools
- R
- ggplot2
- dplyr

## Outcome
The analysis demonstrates how statistical tests can be used to evaluate whether differences between two groups are statistically significant and how the results can inform data-driven decisions.
